1. PROJECT TITLE & DESCRIPTION
--------------------------------------------------------------------------------

Title: A Real-Time Sign language Recognition System

SignBridge is a real-time American Sign Language (ASL) finger-spelling
recognition system that detects hand gestures through a standard webcam and
converts them into text and speech. It uses Google's MediaPipe framework to
extract 21 3D hand landmarks per frame, normalises them relative to the wrist
to achieve position-invariant predictions, and feeds the resulting 63-dimensional
feature vector into a trained Random Forest classifier to identify signed letters
with live confidence scoring. Recognised letters are assembled into words through
a 1.5-second dwell-timer mechanism, and completed words are spoken aloud via the
eSpeak text-to-speech engine — making the system a practical, low-latency
assistive communication tool that runs entirely on edge hardware without any
cloud dependency.

4. SYSTEM OVERVIEW (For Reference)
--------------------------------------------------------------------------------

  Tech Stack  : Python 3.8+, OpenCV, MediaPipe, scikit-learn, NumPy,
                pandas, matplotlib, seaborn, eSpeak, pickle

  Input       : Webcam (V4L2, 640×480 @ 30 FPS)
  Features    : 63-dimensional wrist-relative hand landmark vector
                (21 landmarks × x, y, z)
  Classifier  : Random Forest (100 estimators, random_state=42)
  Threshold   : 75% confidence required to accept a prediction
  Dwell Time  : 1.5 seconds of stable hold to auto-commit a letter
  TTS Engine  : eSpeak (140 WPM, English, non-blocking subprocess)
  Deployment  : Fully offline, edge-only — no cloud API dependency
